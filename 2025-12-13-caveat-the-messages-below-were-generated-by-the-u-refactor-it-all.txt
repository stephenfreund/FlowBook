  Key Simplifications

  1. Single Access Model

  Instead of three separate tracking systems, ONE event stream:
  @dataclass
  class AccessEvent:
      variable: str
      access_type: Literal["READ", "WRITE"]
      detail_type: Optional[Literal["column", "attr"]] = None
      detail_value: Optional[str] = None  # column name or attr name

  2. No Structure-Using vs Structure-Revealing

  Just track ALL accesses. The conflict resolver decides what matters:
  - df['x'] → READ(df, column='x')
  - df.columns → READ(df, attr='columns')
  - df.mean() → READ(df) (no detail - just variable-level read)

  The complexity of "should this count as structural?" moves to the conflict table, which is easy to read and modify.

  3. No Modes

  - Violations (modifying a column someone read) always block
  - Warnings (structural changes affecting structural reads) always warn
  - No configuration needed

  4. Declarative Conflict Rules

  The conflict table is:
  - Readable: Anyone can understand what causes violations
  - Modifiable: Add a row to handle a new case
  - Testable: Each row is a unit test case

  3. Conflict Rules as Proper Models

  from enum import Enum
  from typing import Type, Tuple, Optional, FrozenSet


  class ConflictSeverity(str, Enum):
      """The severity/type of a conflict."""
      OK = "ok"                    # No conflict
      WARNING = "warning"          # Structural warning (depends on mode)
      VIOLATION = "violation"      # Hard violation, always blocked


  class StructuralBehavior(str, Enum):
      """How to treat structural conflicts - maps to user's mode setting."""
      STRUCTURAL = "structural"  # Severity depends on structural_mode setting


  class ConflictRule(BaseModel):
      """
      A declarative rule for detecting conflicts between changes and reads.
      
      Rules are evaluated in order. First matching rule determines the result.
      """

      # What types of changes this rule applies to
      change_types: Tuple[Type[Change], ...]

      # What types of reads this rule applies to  
      read_types: Tuple[Type[ReadEvent], ...]

      # For column-specific rules: require the column names to match?
      same_column: bool = False

      # For structural reads: only match specific attributes?
      # If None, matches any structural attribute
      structural_attrs: Optional[FrozenSet[str]] = None

      # The result when this rule matches
      # Use StructuralBehavior.STRUCTURAL for mode-dependent results
      severity: ConflictSeverity | StructuralBehavior

      # Human-readable description for documentation and error messages
      description: str

      class Config:
          frozen = True

      def matches(self, change: Change, read: ReadEvent) -> bool:
          """Check if this rule applies to the given change/read pair."""
          # Must be correct types
          if not isinstance(change, self.change_types):
              return False
          if not isinstance(read, self.read_types):
              return False

          # Check column matching if required
          if self.same_column:
              change_col = getattr(change, 'column', None)
              read_col = getattr(read, 'column', None)
              if change_col is None or read_col is None:
                  return False
              if change_col != read_col:
                  return False

          # Check structural attribute matching if specified
          if self.structural_attrs is not None:
              if not isinstance(read, StructuralRead):
                  return False
              if read.attr not in self.structural_attrs:
                  return False

          return True

  5. The Conflict Resolver

  from dataclasses import dataclass
  from typing import List


  class StructuralMode(str, Enum):
      """User-configurable mode for structural conflict handling."""
      OFF = "off"        # Ignore structural conflicts entirely
      WARN = "warn"      # Generate warnings but don't block
      ENFORCE = "enforce"  # Treat as violations


  @dataclass
  class ConflictResult:
      """Result of checking one change against one read."""
      severity: ConflictSeverity
      change: Change
      read: ReadEvent
      rule: ConflictRule
      message: str


  class ConflictResolver:
      """
      Resolves conflicts between changes and prior reads using declarative rules.
      
      Usage:
          resolver = ConflictResolver(structural_mode=StructuralMode.WARN)
          
          for change in changes:
              for prior_cell in earlier_cells:
                  for read in prior_cell.reads:
                      result = resolver.resolve(change, read)
                      if result.severity == ConflictSeverity.VIOLATION:
                          # Block execution
                      elif result.severity == ConflictSeverity.WARNING:
                          # Show warning
      """

      def __init__(
          self, 
          structural_mode: StructuralMode = StructuralMode.WARN,
          rules: list[ConflictRule] = CONFLICT_RULES
      ):
          self.structural_mode = structural_mode
          self.rules = rules

      def resolve(self, change: Change, read: ReadEvent) -> ConflictResult:
          """
          Determine if a change conflicts with a prior read.
          
          Args:
              change: A detected change from the current cell
              read: A read event from a prior cell
              
          Returns:
              ConflictResult with severity, matched rule, and message
          """
          # Different variables never conflict
          if change.variable != read.variable:
              return ConflictResult(
                  severity=ConflictSeverity.OK,
                  change=change,
                  read=read,
                  rule=DEFAULT_RULE,
                  message="Different variables"
              )

          # Find first matching rule
          for rule in self.rules:
              if rule.matches(change, read):
                  severity = self._resolve_severity(rule.severity)
                  message = self._format_message(change, read, rule)
                  return ConflictResult(
                      severity=severity,
                      change=change,
                      read=read,
                      rule=rule,
                      message=message
                  )

          # No rule matched - default to OK
          return ConflictResult(
              severity=ConflictSeverity.OK,
              change=change,
              read=read,
              rule=DEFAULT_RULE,
              message="No conflict rule matched"
          )

      def _resolve_severity(
          self, 
          severity: ConflictSeverity | StructuralBehavior
      ) -> ConflictSeverity:
          """Convert StructuralBehavior to ConflictSeverity based on mode."""
          if severity == StructuralBehavior.STRUCTURAL:
              if self.structural_mode == StructuralMode.ENFORCE:
                  return ConflictSeverity.VIOLATION
              elif self.structural_mode == StructuralMode.WARN:
                  return ConflictSeverity.WARNING
              else:  # OFF
                  return ConflictSeverity.OK
          return severity

      def _format_message(
          self, 
          change: Change, 
          read: ReadEvent, 
          rule: ConflictRule
      ) -> str:
          """Generate human-readable conflict message."""
          # Build descriptive message based on types
          change_desc = self._describe_change(change)
          read_desc = self._describe_read(read)

          return f"{change_desc} conflicts with prior {read_desc}. ({rule.description})"

      def _describe_change(self, change: Change) -> str:
          if isinstance(change, ColumnAdded):
              return f"Adding column '{change.column}' to {change.variable}"
          elif isinstance(change, ColumnModified):
              return f"Modifying {change.variable}['{change.column}']"
          elif isinstance(change, ColumnRemoved):
              return f"Removing column '{change.column}' from {change.variable}"
          elif isinstance(change, RowsAdded):
              return f"Adding {change.count} row(s) to {change.variable}"
          elif isinstance(change, RowsRemoved):
              return f"Removing {change.count} row(s) from {change.variable}"
          elif isinstance(change, ValueChanged):
              return f"Changing value of {change.variable}"
          else:
              return f"Modifying {change.variable}"

      def _describe_read(self, read: ReadEvent) -> str:
          if isinstance(read, ColumnRead):
              return f"read of {read.variable}['{read.column}']"
          elif isinstance(read, StructuralRead):
              return f"read of {read.variable}.{read.attr}"
          else:
              return f"read of {read.variable}"

      def check_all(
          self,
          changes: List[Change],
          prior_reads: List[ReadEvent]
      ) -> List[ConflictResult]:
          """
          Check all changes against all prior reads.
          
          Returns list of non-OK results (violations and warnings).
          """
          results = []
          for change in changes:
              for read in prior_reads:
                  result = self.resolve(change, read)
                  if result.severity != ConflictSeverity.OK:
                      results.append(result)
          return results

                      reads.append(ColumnRead(variable=var, column=col))
              if var in tracking.structural_reads:
                  for attr in tracking.structural_reads[var]:
                      reads.append(StructuralRead(variable=var, attr=attr))

              # If no specific reads, add variable-level read
              if (var not in tracking.column_reads_before_writes and
                  var not in tracking.structural_reads):
                  reads.append(VariableRead(variable=var))

          # Convert variable writes
          for var in tracking.writes:
              if var in tracking.column_writes:
                  for col in tracking.column_writes[var]:
                      writes.append(ColumnWrite(variable=var, column=col))
              else:
                  writes.append(VariableWrite(variable=var))

          # Capture structural values
          structural_values = {}
          for var, attrs in tracking.structural_reads.items():
              obj = namespace.get(var)
              if obj is not None:
                  structural_values[var] = {}
                  for attr in attrs:
                      try:
                          structural_values[var][attr] = repr(getattr(obj, attr))[:80]
                      except Exception:
                          structural_values[var][attr] = "<unavailable>"

          return cls(
              cell_id=cell_id,
              execution_seq=execution_seq,
              reads=reads,
              writes=writes,
              structural_values=structural_values
          )

  7. Change Detector (from Diff)

  class ChangeDetector:
      """
      Converts DiffResult to list of typed Change objects.
      
      This extracts semantic meaning from the raw diff.
      """

      def detect(self, diff_result: 'DiffResult') -> List[Change]:
          """Convert diff result to list of Changes."""
          changes = []

          for var_name, diff_node in diff_result.differences.items():
              changes.extend(self._analyze_diff_node(var_name, diff_node))

          return changes

      def _analyze_diff_node(
          self, 
          var_name: str, 
          node: 'DiffNode'
      ) -> List[Change]:
          """Analyze a diff node and extract typed changes."""
          changes = []

          if isinstance(node, ValueComparison):
              # Simple value change
              changes.append(ValueChanged(variable=var_name))

          elif isinstance(node, CompoundDiff):
              if node.source_type == 'dataframe':
                  changes.extend(self._analyze_dataframe_diff(var_name, node))
              elif node.source_type == 'series':
                  changes.extend(self._analyze_series_diff(var_name, node))
              else:
                  # Generic compound change
                  changes.append(ValueChanged(variable=var_name))

          return changes

      def _analyze_dataframe_diff(
          self, 
          var_name: str, 
          node: CompoundDiff
      ) -> List[Change]:
          """Extract DataFrame-specific changes from diff."""
          changes = []

          for key, child in node.children.items():
              # Column-level changes look like "['column_name']" or "['col'][idx]"
              col_match = re.match(r"\['([^']+)'\]", key)
              if col_match:
                  col_name = col_match.group(1)

                  if isinstance(child, ValueComparison):
                      if 'missing' in (child.message or '').lower():
                          if 'pre-state' in (child.message or ''):
                              changes.append(ColumnAdded(variable=var_name, column=col_name))
                          else:
                              changes.append(ColumnRemoved(variable=var_name, column=col_name))
                      else:
                          changes.append(ColumnModified(variable=var_name, column=col_name))
                  else:
                      changes.append(ColumnModified(variable=var_name, column=col_name))

              # Structural changes
              elif key == '._shape':
                  # Analyze shape change to determine row/column adds
                  if isinstance(child, ValueComparison):
                      old_shape = child.value1
                      new_shape = child.value2
                      if old_shape and new_shape:
                          old_rows, old_cols = old_shape
                          new_rows, new_cols = new_shape
                          if new_rows > old_rows:
                              changes.append(RowsAdded(
                                  variable=var_name,
                                  count=new_rows - old_rows
                              ))
                          elif new_rows < old_rows:
                              changes.append(RowsRemoved(
                                  variable=var_name,
                                  count=old_rows - new_rows
                              ))

              elif key == '._index':
                  changes.append(IndexChanged(variable=var_name))

          return changes

  The problem:

  The current system handles this with:

  2. _structure_using_context() - context manager that sets a thread-local flag
  3. Structural tracking checks this flag and skips recording when inside the context

  The new design needs this too. Here's how to incorporate it:

  # =============================================================================
  # ACCESS EVENT EMISSION (not just the events themselves)
  # =============================================================================

  class AccessRecorder:
      """
      Records access events during cell execution.
      
      Handles the structure-using vs structure-revealing distinction:
      - Structure-revealing (df.columns, df.shape): Records StructuralRead
      - Structure-using (repr(df), df['x'], df.mean()): Does NOT record StructuralRead
        even though these internally access structural attributes
      """

      def __init__(self):
          self._events: List[AccessEvent] = []
          self._in_structure_using_context = threading.local()

      @contextmanager
      def structure_using_context(self):
          """
          Context for methods that USE structure but shouldn't RECORD structural reads.
          
          Examples: __repr__, __getitem__, __setitem__, mean(), sum(), etc.
          """
          prev = getattr(self._in_structure_using_context, 'active', False)
          self._in_structure_using_context.active = True
          try:
              yield
          finally:
              self._in_structure_using_context.active = prev

      def record_structural_read(self, variable: str, attr: str):
          """
          Record a structural attribute access - BUT only if we're not inside
          a structure-using method.
          """
          if getattr(self._in_structure_using_context, 'active', False):
              # Inside repr(), __getitem__, etc. - don't record
              return

          self._events.append(StructuralRead(variable=variable, attr=attr))

      def record_column_read(self, variable: str, column: str):
          """Record a column read - always recorded."""
          self._events.append(ColumnRead(variable=variable, column=column))

      def record_column_write(self, variable: str, column: str):
          """Record a column write - always recorded."""
          self._events.append(ColumnWrite(variable=variable, column=column))

  The distinction in the design doc:

  ACCESS RECORDING RULES
  ======================

  Structure-REVEALING methods (record StructuralRead):
    - df.columns, df.shape, df.index, df.dtypes
    - df.keys(), len(df), df.empty
    - df.describe(), df.info(), df.to_dict()
    
    These are recorded because the user EXPLICITLY asked for structural info.

  Structure-USING methods (do NOT record StructuralRead):
    - df['col'], df.loc[], df.iloc[]  (item access)
    - repr(df), str(df), df._repr_html_()  (display)
    - df.mean(), df.sum(), df.apply()  (computation)
    - df + other, df == other  (operators)
    - df.merge(), df.groupby()  (transforms)

    These internally access .columns/.index but the user's PURPOSE
    is not to inspect structure. Recording would cause false positives.

  Implementation: Wrap structure-using methods to set a context flag.
                 Structural attribute getters check the flag before recording.
