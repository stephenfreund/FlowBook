  Summary

  Current bugs:
  1. self.update(initial_ns) bypasses __setitem__ - initial vars not
  mirrored
  2. Missing overrides: get(), setdefault(), pop(), update()
  3. Two dicts can desync

  Recommended: Option D - Delegation Pattern

  class TrackingDict(dict):
      def __init__(self, real_ns):
          super().__init__()  # Empty - we delegate storage
          self._real_ns = real_ns  # This IS user_global_ns

      def __setitem__(self, key, value):
          self._real_ns[key] = value  # Write directly to globals

  Why this is cleaner:
  - Single source of truth: All storage in user_global_ns
  - No mirroring: Writes go directly to the real namespace
  - Python scoping works: List comprehensions find vars because they're in
  user_global_ns
  - Type compatible: Still isinstance(user_ns, dict)

  The key insight: instead of copying data and mirroring writes, we delegate
   all storage to user_global_ns. The TrackingDict is just a tracking
  wrapper, not a separate storage location.
